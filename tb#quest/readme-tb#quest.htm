Introduction

This mod is a collection of small standalone fights (I decided to call the mod 'Quest Pack' because it sounds cooler than 'Fight Pack' or 'Battle Pack'). All of these were written by me (with the usual file which was borrowed from some other mod, as duly noted in the 'thanks' section).

Note: please note that the "Mprolla" components are fourth-wall breaking, so it's not advised to install them, unless you're interested in some non-sense or want to laugh a little at Mprolla.

Compatibility notes

There are no specific install order requests, so install this mod before all mods that are listed as needing to be installed last.
Also, you should install the High Level Revision component of the Refinements mod (see the relevant web site, since some of these components use the HLAs from that mod, and will not work without it. I have no way to enforce this on you (since Refinements should be installed after this mod), so you'll have to be careful yourself.

Installation

Windows

The Bigg Quest Pack is packaged and installed with WeiDU and is distributed as a self-extracting archive. To install, simply double-click the archive and follow the instructions on screen.

Alternatively the files can be extracted from the archive using WinRAR. If properly extracted, you should have a tb#quest folder, and Setup-tb#quest.exe in your BG2 folder. To install, simply double-click Setup-tb#quest.exe and follow the instructions on screen.

Please re-run Setup-tb#quest.exe in your BG2 folder to reinstall, uninstall or otherwise change components.

Macintosh

If the mod was previously installed, uninstall it before extracting the new version. Please note that this mod is incompatible with Mod Organizer. The Bigg Quest Pack is packaged and installed with WeiDU. To install, simply extract the contents of the mod into your BG2 folder. If properly extracted, you should have a tb#quest folder, Setup-tb#quest, and Setup-tb#quest.command in your BG2 folder. To install, simply double-click Setup-tb#quest.command and follow the instructions on screen.

The Bigg Quest Pack > Installation BACK TO TOP
Linux

Download the latest version of WeiDU for Linux from weidu.org and extract (at least) the WeiDU, weinstall and tolower executables in a $PATH directory (E.G. /usr/bin).
Following that, open a terminal, cd to your BG2 installation directory, run `tolower' and answer Y to both queries.
You can avoid running the second option (linux.ini) if you've already ran it once in the same directory. To save time, the archive is already tolowered, so there's no need to run the first option (lowercasing file names) either if you've extracted only this mod since the last time you lower cased file name.
If you're unsure, running tolower and choosing both options is the safe bet.
After that, run 'weinstall tb#quest' from your BG2 installation directory to install or uninstall the mod.

The Curse of Mprolla the Spammer

WARNING: FOURTH WALL BREAKING COMPONENT
WARNING: CHEESE
With this component installed, you can meet (and defeat) Mprolla, to avenge all the years of spamming and trolling he had you sustain in real life. He can be found in the Waukeen's Promenade, SE of Radeel, x4916 y601. Note: this component includes a spell from Refinements (his improved Jester Song).

The Curse of Mprolla the Cheater

An even cheesier version of the above component. (as such, it displays only if you have installed the above component).

Large Scale Battle

You (with the help of some Amnish Soldiers) will infiltrate in an army's camp and route them by killing their leaders. The battlefield contains one hundred twenty-six enemies (and, of course, I will spawn an hectic amount of extra soldiers for you to slice), most of them are orcs, but there is a good number of higher-level guys.
High Level Abilities (where possible) are chosen from the Refinements pool; this means that Refinements is needed.
You get also a new spell and a new item (the scroll teaching it, that is...) by fighting the battle.
The mini-quest will trigger when your character reaches 2.500.000 XP points and you are outside; if you want to try the fight before this XP level, you can do this via the console: the needed commands are CLUAConsole:CreateCreature("tb#lotr") and CLUAConsole:SetGlobal("tb#spawnedlotr","global",1)

Underdark Mage Duel

In the Drow City in the Underdark, you meet a nameless drow wizard near the tower of Deirex (he uses the elf avatar); if a male wizard talks to him, he might say (at random) that he's afraid you came to take his place in Sorcere. With this component installed, you always get that line if you talk to him (his name is now Laniz) with either a mage (pure or multiclass), a bard or a sorceror, and he will challenge you to a duel. You can accept to duel him.
Well, that's it :)
